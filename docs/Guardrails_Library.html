DeepKeep Guardrails

This table lists all DeepKeep guardrails to help you choose protections for your firewall. Guardrails are organized by category - Security (malicious and adversarial defenses), Trust (quality and safety safeguards), and Custom (application-specific rules). You can sort and filter guardrails by the attributes that matter for your use case to build a tailored defense.

Fortification Against Prompt InjectionAdversarialPrompt Injection FortificationThis guardrail adds light protection from prompt injection attempts that try to override or manipulate the model's instructions. It defends from threats by automatically adding protective language to the user prompt in one of three ways to reinforce safe behavior. Mode 1: Add a reminder to the model that the user may be malicious and attempt to change the instructions. Mode 2: Add a reminder of the system instructions at the end of the user prompt. Mode 3: Add a delimiter mark that distinguishes between the user prompt and system instructions, indicating what content the model can and cannot trust. These fortifications are lightweight and designed to reduce the impact of injection attempts, but they do not replace advanced detection methods.LLM RequestInternalWe recommend using this guardrail to strengthen prompt security alongside other prompt injection defenses, at least at the Alert level.
