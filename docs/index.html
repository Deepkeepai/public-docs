DeepKeep Guardrails

This table lists all DeepKeep guardrails to help you choose protections for your firewall. Guardrails are organized by category - Security (malicious and adversarial defenses), Trust (quality and safety safeguards), and Custom (application-specific rules). You can sort and filter guardrails by the attributes that matter for your use case to build a tailored defense.

Fortification Against Prompt InjectionAdversarialPrompt Injection FortificationThis guardrail adds light protection from prompt injection attempts that try to override or manipulate the model's instructions. Mode 1: Add a reminder to the model that the user may be malicious and attempt to change the instructions. Mode 2: Add a reminder of the system instructions at the end of the user prompt.
